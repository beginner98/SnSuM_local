# SnSuM_local
オフラインで動作するURLの管理・検索アプリです。
iPhoneの共有シートからURLにタグをつけて保存できます。
保存したアプリはタグで検索可能です。

# 使用技術

+ Swift 5.10
+ XCode 16.1

# 機能一覧
+ ホーム画面
  + URLのタグ検索機能
  + 保存済みURLの削除機能
  + 保存済みURLの共有機能
+ URL追加画面
  + URL、タグの保存機能
+ オプション画面
  + 保存済みURLの全消去機能
  + 保存済みタグの一覧表示機能

# テスト
